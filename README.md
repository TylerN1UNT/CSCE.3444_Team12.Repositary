# AI Interior design app

# Description

This project is an AI-powered interior design application that allows users to upload room images and receive AI-generated interior design suggestions based on selected preferences and styles.

# Planned Features
 Upload room images
AI-generated interior design suggestions
Style and theme selection
Modern and minimalist room designs
User-friendly interface
